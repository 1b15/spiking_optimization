Sparse Coding in V1

example dictionary $\mathbf{\Phi}$

Problem: reconstruct visual input $\mathbf{x}$
from overcomplete dictionary $\mathbf{\Phi}$
with sparse activation of neurons $\mathbf{a}$

$ \text{min}\|\mathbf{a}\|_1 $, such that
$ \|\mathbf{x} - \mathbf{\Phi}\mathbf{a}\|_2 = 0 $

This is equivalent to LASSO:
$ \text{min} \{ \|\mathbf{x} - \mathbf{\Phi}\mathbf{a}\|_2^2 + \lambda \|\mathbf{a}\|_1 \} $

... and can be solved with the dynamics
$ \dot{u_i}(t) = b_i - u_i(t) - \sum_{j\neq i}w_{ij}T(u_j(t)) $
with driving term $b_i = \langle \mathbf{\Phi}_i, \mathbf{x}(t) \rangle$,
thresholding function $T$, and $a_i = T(u_i)$
