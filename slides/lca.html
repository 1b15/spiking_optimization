Sparse Coding in V1

example dictionary $\mathbf{\Phi}$

Problem: reconstruct visual input $\mathbf{x}$
from overcomplete dictionary $\mathbf{\Phi}$
with sparse activation of neurons $\mathbf{u}$

$ \text{min}\|\mathbf{u}\|_1 $, such that
$ \|\mathbf{x} - \mathbf{\Phi}\mathbf{u}\|_2 = 0 $

This is equivalent to LASSO:
$ \text{min} \{ \|\mathbf{x} - \mathbf{\Phi}\mathbf{u}\|_2^2 + \lambda \|\mathbf{u}\|_1 \} $

... and can be solved with the dynamics
$ \dot{u_i}(t) = b_i - u_i(t) - \sum_{j\neq i}w_{ij}T(u_j(t)) $
with driving term $b_i$ and thresholding function $T$
